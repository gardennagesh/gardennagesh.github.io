TABLE Operation CMDS:
---------------------------
CMDS -- MEANING
1) t or T -- Create Table
2) r or R -- Remove Table
3) c or C -- Column Add to existing table
4) o or O -- Column remove from existing table

ROW ReadWrite CMDS:
--------------------------
1) i or I -- Insert a new row to table (By default at last)
2) a or A -- Append a row to existing table.
3) u or U -- Update a row
4) s or S -- Select rows from table
5) f or F -- Find a row from table
6) d or D -- Delete a row from table

DATA TYPES:
-----------------------
int or INT - Integer type
float or FLOAT - float type
char or CHAR - char type
ALSTR - Alphbet string type. Eg APSTR26 can hold string of length 26.
ANSTR - Alphanumeric string type. Eg ANSTR9 can hold 9 alphanumeric string.
DATE: ddmmyyyy
TIME: HHMMSS
BOOL: true or false

CONDITIONS:
--------------------------
eq, ne, gt, lt, ge, le ( For comparing int or strings of equal, not-equal, greater, lesser)
and, or, not (To join conditions).
? . Question mark in statement indicates that there is condion succeeded.

COMMAND FORMAT:
--------------------------
"stamanet" or 'statement' or [statment]
Statement should start and end with " ", or ' ' or [ ]. For now [statement]
For now, Statement should be one line.
"Primery Key" must be in First column and last column is for "Forein key" .

COMAND SYNTAX:
-----------------------
[CMD : TABLENAME: col1_name.datatype : col2_name.datatype : col3_name.datatype: ...]

[CMD : TABLENAME: c1.value1 : c2.value2 : c3.value3 : ...]

[CMD : TABLENAME.col_name1 : TABLENAMe.col_name2 ? CONDITION]

Examples:
Lets create a PPLGREAT Table using CMD T

[T : PPLGREAT : PPLID.INT : BIRTHDAY.DATE : NAME.ALSTR20 : WORK.ANSTR45] .

Above statement creates a table PPLGREAT with cmd T.
